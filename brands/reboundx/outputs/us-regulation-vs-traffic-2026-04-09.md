- **기관:** 상원 (Tillis R / Alsobrooks D)
- **내용:** 스테이블코인 수동 잔액 이자 지급 허용 여부에 대한 타협안 협상 시작. 수동 잔액 보상 금지 방향 검토. DeFi 처리 방식·불법 금융 조항 미합의.
- **영향도:** 🟡 중간 | **시장 함의:** 업계 최우선 법안 입법 진전 긍정 신호.

### 2026-03-11 | SEC-CFTC 역사적 MOU 체결
- **기관:** SEC (Atkins 위원장) + CFTC (Selig 위원장)
- **내용:** 6개 영역에서 규제 관할 공동 조율 합의 — 제품 정의, 청산·마진, 이중 등록, 디지털 자산 프레임워크, 보고 통합, 공동 감시. "최소 유효 용량(minimum effective dose)" 철학 채택.
- **영향도:** 🔴 매우 높음 | **시장 함의:** SEC-CFTC 관할 분쟁 공식 종식. 친혁신 통일 기조 확립.

### 2026-03-17 | SEC-CFTC 공동 68페이지 크립토 자산 분류 해석 발표 (역사적 이정표)

- **영향도:** 🔴 매우 높음 | **시장 반응:** BTC $75,000 횡보(매크로 압박). 비트코인 ETF 순유입 약 $55억 (4개월 만에 반전). XRP ETF $14.4억, SOL ETF 주간 $110만 순유입.

## 4. 규제 이벤트 대응 콘텐츠 템플릿

### 템플릿 A — SEC/CFTC 발표 직후 (명확화·우호적 뉴스)

**제목:** "SEC가 크립토 규제를 명확히 했습니다 — 지금 수수료를 절감하는 가장 안전한 방법"

**구조:**
```
[훅] SEC가 방금 [이벤트 요약]을 발표했습니다.
     규제가 명확해진 지금, 더 이상 불확실성을 이유로
     높은 수수료를 낼 필요가 없습니다.

[교육] 새 규제 하에서 달라지는 것 3가지
  1. [분류 변경 또는 허용 범위 확대]
  2. [컴플라이언스 요건 변화]
  3. [사용자 보호 강화 내용]

[브릿지] 저희가 이 변화에 어떻게 대응했는지

[CTA] 지금 수수료 비교하기 (투자 권유 없음, 정보 제공 목적)
```

**주의:** 수익 보장·투자 권유 문구 없음. "안전하게"는 컴플라이언스 의미로만 사용.

```
[행동 안내] 사용자가 지금 할 수 있는 것 (있는 경우)

[신뢰 CTA] 컴플라이언스 상세 페이지 보기 / 고객센터 문의
```

**주의:** "안전한 투자처"류 문구 금지. 사실 기반 현황 공개에 집중.

### 템플릿 C — 법안 진행 중 (불확실성 해소형)

**제목:** "상원 크립토 법안, 지금 어디까지 왔나요? — 알아두면 유리한 타임라인 정리"

**구조:**
```
[현황 요약] 현재 법안 진행 단계 (위원회 → 마크업 → 본회의)

[타임라인] 주요 일정과 예상 결과 (확정 정보 + 불확실한 부분 명시)

[시나리오별 의미]
  - 법안 통과 시: 사용자에게 달라지는 점
  - 수정·지연 시: 현행 유지 내용

[우리의 대응] 어떤 시나리오에도 대비하는 서비스 구조

[CTA] 법안 업데이트 알림 받기 (뉴스레터/알림 구독)
```

**주의:** 법안 결과 예측 단정 금지. "예상됩니다", "가능성이 있습니다" 표현 사용.

## 5. 수익 보장/투자 권유 표현 체크리스트 (업데이트)

### 5-1. 절대 금지 표현

| 금지 표현 | 금지 이유 | 대체 표현 |
|----------|----------|----------|
| "수익이 보장됩니다" | 투자 권유, 허위 광고 | 사용 불가 (대체 없음) |
| "연 X% 수익 예상" | 미래 수익 보장 암시 | "과거 데이터 기준 (미래 보장 아님)" |
| "지금 투자하세요" | 직접 투자 권유 | "지금 알아보세요" / "비교해보세요" |
| "손실 없는 투자" | 현실 왜곡 | 사용 불가 |
| "규제 리스크 없음" | 허위 사실 | "현재 규제 동향 확인 가능" |
| "안전한 투자처" | 투자 권유 + 보장 암시 | "컴플라이언스 기준 충족 서비스" |
| "지금 사야 할 코인" | 직접 투자 권유 | 사용 불가 |
| "수수료 절감 = 수익 증가" | 수익 보장 암시 | "수수료 절감으로 비용 최적화" |

### 5-2. 조건부 허용 표현 (컨텍스트 명시 필수)

| 표현 | 허용 조건 |
|------|----------|
| "수수료 절감" | 실제 수수료 비교 데이터 첨부 시 허용 |
| "비용 효율적" | 특정 기능·서비스 범위 명시 시 허용 |
| "안전하게" | 컴플라이언스·보안 맥락에서만 허용 (투자 안전성 아님) |
| "성장하는 시장" | 통계 출처 명시 + 미래 보장 아님 고지 시 허용 |
| "시장 대비 낮은 수수료" | 비교 기준 시점·대상 명시 시 허용 |

### 5-3. 규제 이벤트 발생 시 추가 주의 표현

> AML·제재 관련 발표 직후 일시적으로 더 강한 주의 필요

- "규제가 바뀌어도 수익은 유지됩니다" → **즉시 삭제** (허위 안심 제공)
- "새 규제 덕분에 더 안전한 투자" → **수정 필요** → "새 규제로 컴플라이언스 기준 강화"
- "정부 승인 플랫폼" → 공식 인가 없을 경우 **절대 사용 금지**
- "[규제 기관명] 인정" → 실제 공식 인정 문서 없을 경우 **절대 사용 금지**

### 5-4. 콘텐츠 발행 전 최종 체크리스트

```
□ 투자 권유 문구 포함 여부 확인
□ 수익 보장·예측 수치 포함 여부 확인
□ "안전한 투자" 류 복합 표현 확인
□ 규제 기관 공식 입장 인용 시 출처 링크 첨부
□ 규제 현황 설명 시 "정보 제공 목적, 투자 조언 아님" 고지
□ 스테이블코인/DeFi 관련 콘텐츠 → GENIUS Act 관련 면책 문구 추가
□ 최신 규제 이벤트 반영 여부 확인 (이 파일 타임라인 참조)
```
